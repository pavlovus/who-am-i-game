# Design System: Midnight Green

## Бібліотеки
- JavaFX (`javafx-controls`, `javafx-fxml`)
- AtlantaFX (базова тема `PrimerDark`)
- ControlsFX (для додаткових компонентів та сповіщень)

## Стилістика
**Midnight Green** — преміальний темний інтерфейс із смарагдовими акцентами.

## Кольори
- **Фон сторінки:** `#0a0f0d`
- **Фон картки/панелі:** `#111a14`
- **Border карток:** `#1a2e1f`
- **Акцент primary:** `#00d46a`
- **Акцент hover:** `#00f077`
- **Акцент danger:** `#f85149`
- **Акцент warning:** `#d29922`
- **Текст основний:** `#e6edf3`
- **Текст другорядний:** `#8b949e`
- **Glow ефект (тінь акценту):** `#00d46a55`

## Форми та відступи
- **Border-radius карток:** `12px`
- **Border-radius кнопок:** `8px`
- **Border-radius полів вводу:** `8px`
- **Padding карток:** `24px`
- **Gap між елементами:** 
  - Малий: `12px`
  - Середній: `24px`
  - Великий: `40px`

## Анімації (JavaFX Timeline / FadeTransition)
- **Fade переходи:** `150ms ease-in-out`
- **Hover на кнопках:** `100ms`
- **Shake при помилці:** `300ms`
- **Glow пульсація на акцентних елементах:** `1500ms` циклічно

## Кнопки
- **Primary:** фон `#00d46a`, темний текст `#0a0f0d`, hover фон `#00f077` + легкий glow. Використовувати класи `.button-primary` та налаштування через CSS/код.
- **Danger:** фон `#f85149`, білий текст, hover +10% яскравість.
- **Ghost:** прозорий фон, border `#1a2e1f`, hover фон `#111a14`.
- **Висота кнопок:** `40px` (звичайна), `48px` (велика).

## Типографіка
- **H1 заголовок:** Bold `32px`
- **H2 заголовок:** Bold `24px`
- **H3 підзаголовок:** SemiBold `18px`
- **Основний текст:** Regular `14px`
- **Другорядний текст:** Regular `12px`, колір `#8b949e`

## Заборонено
- Зовнішні шрифти та іконки-бібліотеки.
- Захардкоджені розміри через `setPrefWidth/Height` там, де можна використати `Priority.ALWAYS` (використовувати VBox/HBox).
- Використання `module-info.java`.
